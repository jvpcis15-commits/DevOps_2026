# DevOps_2026
Trabalho de DevOps

Este projeto iniciou-se como somente como uma demonstraçãode data e hora mas acabou consistindo-se no desenvolvimento de uma calculadora em Python, criada com o objetivo de aplicar funções, estruturas de repetição e controle de versão utilizando o GitHub.

O sistema permite realizar operações matemáticas básicas, além de implementar funcionalidades adicionais como histórico de operações e registro de data e hora.

🚀 Funcionalidades

✔ Operações básicas:

Soma
Subtração
Multiplicação
Divisão (com tratamento de erro)

✔ Funcionalidades extras:

Registro de data e hora das operações
Histórico de cálculos realizados
Exibição da última operação
Validação de entrada de dados
Menu interativo no terminal
🛠️ Tecnologias utilizadas
Python 3
Biblioteca padrão datetime
